# Fofo Breakout

### The classic arcade game... with a twist!

The classic arcade game Breakout where the player controls a paddle at the bottom of the screen and bounces a ball on it to destroy all the blocks at the top of the screen now comes with a couple of extra features which will make the original experience more fun... and challenging!

Created by Fofo13az (Adolfo Hernández)

## How to Start
Since this game was created using JS and HTML, you have to run the **breakout.html** file found in this same repository to start playing.

## How to Play

- Press **Space** to serve the ball
- Use **A** and **D** to move the paddle left and right
- Bounce the ball off your paddle to hit and destroy blocks
- If the ball falls below the screen, you lose a life
- Press **R** to restart after a game over

## Rules

- You start with **3 lives**
- Each block destroyed awards **1 point**
- Losing all lives ends the game in a **loss**
- Destroying all blocks ends the game in a **win**

## Objective

Destroy ALL the blocks before running out of lives.

## ...the TWIST!

Fofo Breakout introduces TWO new mechanics to the game, making it more challenging:

1. **Paddle shrinking**: Every time you destroy an entire row of blocks (or the equivalent 8 blocks) your paddle will shrink slightly, until it reaches a minimum size of 50px. This will make it harder for you to keep the ball bouncing!
   
2. **Speed increase**: At the same intervals, both the ball speed and paddle speed gradually increase, giving you less reaction time as the game progresses!

These mechanics ensure the game gets harder the closer you are to winning, creating a fun, challenging and refreshing version of Breakout.
